# Modeling Pipeline

This directory contains a first-pass Python training pipeline for the S-AKI dynamic prediction study.

## Input

The script expects an exported modeling table from MIMIC-IV, eICU, or SICdb.

Minimum required columns:

- `stay_id`
- `landmark_hour`
- `aki_progression_48h`

Recommended columns:

- demographic, vital-sign, laboratory, severity-score, treatment, and missingness predictors generated by the SQL pipeline.

Outcome-derived columns such as hospital death/discharge status and raw time-offset columns are excluded from training by default to reduce leakage risk.

## Basic Usage

Install dependencies first:

```bash
python -m pip install -r requirements.txt
```

```bash
python train_dynamic_model.py \
  --input path/to/modeling_dataset.csv \
  --output-dir results/mimic_iv_first_run \
  --target aki_progression_48h \
  --group-col stay_id
```

## Outputs

- `metrics.csv`: AUROC, AUPRC, Brier score, calibration intercept, calibration slope.
- `predictions.csv`: internal test-set predictions.
- `calibration_bins.csv`: calibration data by decile.
- `decision_curve_*.csv`: decision curve data across risk thresholds.
- `feature_importance_*.csv`: model-specific feature importance when available.
- `model_*.pkl`: saved trained model pipelines for external validation.
- `run_config.json`: command and feature metadata.

## Optional Dependencies

The script always tries to run logistic regression and random forest using scikit-learn.

If available, it also runs:

- XGBoost
- LightGBM

SHAP plots are intentionally not generated in this first-pass script. After the winning model is selected, SHAP should be run in a dedicated notebook or script using the locked feature matrix.

## External Validation

After training, validate the locked model on an external cohort:

```bash
python external_validate.py \
  --input path/to/eicu_modeling_dataset.csv \
  --model results/mimic_iv_first_run/model_lightgbm.pkl \
  --predictors results/mimic_iv_first_run/predictor_columns.json \
  --output-dir results/eicu_external_validation \
  --cohort-name eicu
```

Use the unrecalibrated validation metrics as the primary external validation results. Recalibration, if performed, should be reported separately.
