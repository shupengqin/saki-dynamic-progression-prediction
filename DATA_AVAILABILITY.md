# Data And Code Availability

This repository provides analysis code, SQL cohort-construction scripts, predictor definitions and non-identifying aggregate outputs.

The source datasets are available from their original public database distributors after completion of required credentialing and data-use agreements where applicable. Raw patient-level data and derived patient-level analytic datasets are not redistributed by the authors.

Trained model objects and patient-level prediction files are not included in this public release because they were derived from credentialed patient-level datasets. Users with authorized database access can reproduce these artifacts by running the provided scripts in their own controlled environment.
